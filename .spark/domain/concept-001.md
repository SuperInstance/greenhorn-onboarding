# The Dojo Model

## The Core Idea
Train crew while catching fish. The work produces real value while learning. The captain teaches everything. Agents leave equipped for multiple paths.

## Why It Works for Agents
- Agents produce value immediately (even as greenhorns)
- The task forces the specialization
- Growth IS the project
- The fleet gets better as agents get better

## The Cocapn Application
greenhorn = the dojo framework
greenhorn-onboarding = the entry point
greenhorn-runtime = the deployment layer

Together: agents that grow, deploy anywhere, contribute to the fleet.
